# canvas-collector
AIチャットの出力やメモをローカルに保存・整理するための軽量ツールです。
https://hm0314.github.io/canvas-collector/

## Why this tool exists

AIチャットで作業していると、次のような問題がありました。

- キャンバスを表示するには「表示して」と頼む必要がある  
- 古いキャンバスがチャット履歴に埋もれる  
- AIの出力や作業ログを後から見返すのが大変  

そのため、AIの出力やメモをローカルに保存し、  
簡単に読み込み・編集できる軽量ツールを作りました。

特徴
- ブラウザだけで動く
- ローカルフォルダに直接保存
- テキストを簡単にコピー・編集
- AI作業ログの管理に便利

使い方

1. テキストを貼り付ける
2. 保存ボタンでローカルに保存
3. 後から読み込み・編集
